# Hey, I'm Aria

I work with Solid.js, React, Node.js, and Python most days.

## What I'm working on

[**buddy.nvim**](https://github.com/arismoko/buddy.nvim) - MCP server for Neovim. You write tools in Lua and they work as both MCP tools and native Neovim plugins.

[**LDOC**](https://github.com/arismoko/LDOC) - A plain-text DSL that compiles to DOCX for legal documents. I wrote the whole compiler pipeline in TypeScript (lexer, parser, binder, emitter), added embedded Lua expressions, and built an LSP with completions, diagnostics, and go-to-definition. A personal project I use at work.

[**Playmat**](https://github.com/arismoko/playmat) - Cockatrice in the browser. Multiplayer Magic: The Gathering tabletop built with React and PartyKit. Deck building, card zones, and real-time sync in the browser.

[**Phish Food**](https://arismoko.itch.io/phish-food) - Hand-drawn platformer I built in Unity for SEAJAM. Dynamic music, story-driven, rated 4/5.
